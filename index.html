Conhecimentos

Html (Hyper Text Markup Language) é a linguagem responsavel pela estruturação do site, ela é uma das principais linguagens no desenvolvimento front-end, utilizando tags para fazer toda a estrutura de uma pagina web

CSS (Cascading Style Sheets ou Folhas de Estilo em Cascata) é uma linguagem de estilo usada para estilizar os elementos HTML, alterando suas propriedades como cor de fundo, tamanho de fonte, tipo de fonte entre outras propriedades

O JavaScript é uma linguagem de programação de alto nível que pode ser utilizada tanto no front-end como no back-end. Junto do HTML e do CSS, é uma das principais tecnologias da web, com ela é possível criar páginas interativas com elementos dinâmicos e boa performance

GIT

Git é uma ferramenta de versionamento open source usado para controlar o histórico de alterações de arquivos. Ele permite mais flexibilidade no fluxo de trabalho, segurança e desempenho.

GITHUB

o GitHub é um serviço baseado em nuvem que utiliza o controle de versão Git. Nele os desenvolvedores podem colabor e fazer mudanças em projetos compartilhados enquanto mantêm todo seu progresso registrado detalhadamente.
